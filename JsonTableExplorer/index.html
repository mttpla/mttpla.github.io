What it is

JSON Table Explorer is a browser-only technical viewer for arbitrary JSON. It turns top-level arrays into independent tables, flattens nested objects into columns, and keeps document metadata separate from tabular data.

How to use it

Click Choose File in the JSON Input section and select a JSON file from your system, or click Paste JSON to paste raw JSON data directly. Both methods parse the data entirely in the browser. If the document contains multiple top-level arrays, the explorer creates one table for each array.

What you can do

  • Sort table rows by column
  • Filter data with Global Search
  • Hide columns you do not want to display
  • Inspect flattened nested fields as regular columns
  • Export each table with Export CSV

Settings

Use the Settings panel for compact technical controls over schema, columns and aggregations.

  • Flatten depth: controls how deeply nested objects are expanded into columns. A higher value explores deeper nested paths. Use Apply depth to rebuild the table structure.
  • Page size: controls how many rows are shown per page. Available options are 10, 20, 50 and 100. Use Apply size to update pagination.
  • Visible columns: controls how many detected columns are shown by default when a dataset is loaded. Additional columns remain available in Settings and start hidden.

Metadata and structure

Non-tabular metadata is shown separately, so contextual information is not mixed with table rows. This is useful for JSON documents that include both metadata and one or more data arrays.

Privacy

All processing happens client-side in your browser. JSON files are not uploaded, stored or sent to any server, which makes the tool suitable for inspecting private or sensitive data locally.
